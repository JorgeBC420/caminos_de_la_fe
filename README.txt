# Caminos de la Fe: Cruzada y Conquista

**Un RPG de acción medieval-fantástico ambientado en las Cruzadas.**

## Características Clave
- Acción 3D en tiempo real
- Atributos personalizables con oro y experiencia
- Equipamiento modular y monturas
- Sistema de facciones y reputación
- Armas legendarias e ítems únicos por servidor
- Eventos PvP, misiones épicas, guerras de clanes

## Estructura del Proyecto
- `systems/`: Stats, reputación, combate
- `inventory/`: Inventario general y UI
- `items/`: Armas, armaduras, consumibles, legendarios
- `ui/`: Interfaces de stats, equipo, inventario, facciones
- `quests/`: Misiones por facción y progresión

## Instalación
```bash
pip install -r requirements.txt
python main.py
